# Contributing

Thank you for your interest in contributing to this repository.

## How to contribute

1. Fork the repository.
2. Create a feature branch: `git checkout -b feature-name`
3. Make your changes and run the code locally.
4. Commit your work: `git commit -m "Add feature description"`
5. Push to your fork: `git push origin feature-name`
6. Open a pull request describing your changes.

## Notes

- Keep changes focused and clearly described.
- If you add new Python dependencies, update `requirements.txt`.
- Respect the repository structure and add files to the appropriate folder.
